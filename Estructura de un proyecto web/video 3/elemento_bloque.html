Elemento de bloque vs elemento de linea

Cada elemento HTML tiene un valor de visualización predeterminado,
según el tipo de elemento que sea. Hay dos valores de visualización: bloque y en línea.

Elementos de bloque

Un elemento a nivel de bloque siempre comienza en una nueva línea y
los navegadores automáticamente agregan algo de espacio (un margen) antes y después del elemento.
Un elemento a nivel de bloque siempre ocupa todo el ancho disponible
(se extiende hacia la izquierda y hacia la derecha tanto como puede).
Dos elementos de bloque de uso común son los tags 'p' y 'div'.
El "p" elemento define un párrafo en un documento HTML.
El "div" elemento define una división o una sección en un documento HTML

Elementos de linea

Un elemento en línea no comienza en una nueva línea.
Un elemento en línea solo ocupa el ancho necesario.
Este es un elemento 'span' dentro de un párrafo.
